Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in DOM are ID(getElementById), Class name(getElementsbyClassname), Query Selectors, Tag name.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML will get or modify the html tags of an attribute, the innerText will get or modify the texual content of an element and the innerContent will get the raw text with all the hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JS that allows the parent elemt to handle its events instead of its giving out events to its childs separately.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in JS that starts from a target element and bubbles to the parent element bubbling through the DOM tree triggering all the events in the way.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create an element by using createElement and append a child element by using appendChild, we can setAttribute by using setAttribute and remove by removeAttribute. We can also use removeChild to remove a child element from a parent element.
